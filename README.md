## 프로젝트 개요

**Java (JDBC)** 를 통해 주어진 요구조건에 따라 작동하는 **Console Application** 을 작성한다.

## 프로젝트 요구사항

### **Database** 생성
  - 야구장 테이블 (총 3개 야구장)
  - 팀 테이블 (총 3개 팀)
  - 선수 테이블 (팀당 각 9명)

### **Java Class** 생성
  - BaseballApp 생성
  - DBConnection 생성
  - Model 생성
  - Dao 생성
  - Service 생성

### **기능** 구현
  - 야구장 등록
  - 전체 야구장 목록보기
  - 팀 등록
  - 전체 팀 목록
  - 선수 등록
  - 팀별 선수 목록
  - 선수 퇴출 등록
  - 선수 퇴출 목록
  - 포지션별 팀 야구 선수 출력
